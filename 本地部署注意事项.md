# 本地部署注意事项

## 目录结构

```
j-index-tts/
├── checkpoints/                    # 本地模型目录
│   ├── w2v_bert/                   # → 软链接到 /mnt/e/ai-models/hf_models/facebook/w2v-bert-2.0/
│   ├── bigvgan_v2/                 # → 软链接到 /mnt/e/ai-models/models--nvidia--bigvgan_v2_22khz_80band_256x/
│   ├── MaskGCT_model/              # MaskGCT 模型
│   ├── campplus_cn_common.bin     # → 软链接到 /mnt/e/ai-models/models--funasr--campplus/...
│   ├── gpt.pth                     # GPT 模型 (~3.3GB)
│   ├── s2mel.pth                   # S2Mel 模型 (~1.1GB)
│   ├── feat1.pt                    # Speaker 特征
│   ├── feat2.pt                    # Emotion 特征
│   ├── wav2vec2bert_stats.pt       # Wav2Vec 统计
│   ├── bpe.model                   # BPE 分词模型
│   └── qwen0.6bemo4-merge/         # Qwen Emotion 模型
└── run_optimized.sh               # 优化启动脚本
```

## 软链接说明

项目使用软链接指向 `/mnt/e/ai-models/` 共享模型目录：

| 本地路径 | 指向 | 模型来源 |
|---------|------|---------|
| `checkpoints/w2v_bert/` | `/mnt/e/ai-models/hf_models/facebook/w2v-bert-2.0/` | HuggingFace |
| `checkpoints/bigvgan_v2/` | `/mnt/e/ai-models/models--nvidia--bigvgan_v2_22khz_80band_256x/` | HuggingFace |
| `checkpoints/campplus_cn_common.bin` | `/mnt/e/ai-models/models--funasr--campplus/...` | HuggingFace |

**注意**：确保 `/mnt/e/ai-models/` 目录存在且包含所有模型文件。

## 环境变量

启动脚本中设置的环境变量：

```bash
export HF_HOME="/mnt/e/ai-models"                    # HuggingFace 模型主目录
export HF_ENDPOINT="https://hf-mirror.com"           # HuggingFace 镜像站
export MODELSCOPE_CACHE="/mnt/e/ai-models/ms_cache"  # ModelScope 缓存
export TRANSFORMERS_CACHE="/mnt/e/ai-models/transformers_cache"  # Transformers 缓存
export HF_DATASETS_CACHE="/mnt/e/ai-models/datasets_cache"      # 数据集缓存
```

## 模型文件大小

| 文件 | 大小 | 说明 |
|------|------|------|
| `gpt.pth` | ~3.3 GB | GPT 主模型 |
| `s2mel.pth` | ~1.1 GB | S2Mel 转换模型 |
| `w2v_bert/` | ~3 GB | Wav2Vec BERT 模型 |

| 总计 | ~8 GB+ | 全部模型 |

## 部署前提条件

1. **模型目录就绪**
   - 确保 `/mnt/e/ai-models/` 包含所有模型
   - 或修改 `run_optimized.sh` 中的路径指向你的模型目录

2. **Python 环境**
   ```bash
   uv run webui.py --fp16 --port 7860
   ```

3. **显存要求**
   - FP16 模式: ~8.5 GB
   - 推荐: 10GB+ 显存

## 常见问题

### 1. 软链接失效
如果软链接指向的文件不存在，会报错。检查：
```bash
ls -la checkpoints/w2v_bert/
ls -la checkpoints/bigvgan_v2/
ls -la checkpoints/campplus_cn_common.bin
```

### 2. 模型下载
首次运行如果本地模型不存在，会自动从 HuggingFace 下载：
- 设置 `HF_ENDPOINT="https://hf-mirror.com"` 使用镜像加速
- 或手动下载到 `/mnt/e/ai-models/` 对应目录

### 3. 示例音频未下载
```bash
git lfs pull
```
`examples/` 目录下的音频是 Git LFS 指针。

### 4. 显存不足 (OOM)
- 使用 `--fp16` 半精度
- 不启用 `--deepspeed`
- 不启用 `--cuda_kernel`
